# grocery-data-qc-analysis
Structured Excel QC, validation, and reporting workflow project using a retail grocery sales dataset.

## Objective

Demonstrate a structured Excel-based workflow for:
- data cleaning
- QC validation
- categorical profiling
- normalization
- pivot analysis
- derived metric evaluation

## Workflow Overview

The project includes:
- checksum-style validation
- missing-value identification
- categorical field profiling
- normalization workflows
- derived metric analysis
- structured pivot reporting
- audit-oriented workbook documentation

## Derived Metric

`loyalty_points_per_final_amount`

Used to compare loyalty point behavior relative to total purchase amounts across store locations.

## Workbook Structure

- Appendix A – Data cleaning, QC validation, and checksum workflows
- Appendix B – Categorical profiling and normalization examples
- Appendix C – Pivot analysis and derived metrics

## Files Included

- Excel workbook with supporting appendices
